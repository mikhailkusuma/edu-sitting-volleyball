Peraturan Sitting Volleyball



Sitting volleyball adalah versi olahraga voli yang dimainkan oleh para atlet dengan duduk di atas lantai. Peraturan untuk sitting volleyball sebagian besar mirip dengan peraturan voli tradisional, namun ada beberapa perbedaan yang disesuaikan untuk memfasilitasi permainan para atlet yang duduk. Berikut adalah beberapa peraturan umum untuk sitting volleyball:

1. Tim

  • Setiap tim terdiri dari enam pemain.
  • Pemain harus selalu duduk di atas pantat mereka dan mempertahankan kontak dengan lantai.
2. Permainan dan Set
  • Permainan terdiri dari dua set pertama hingga 25 poin dan set ketiga hingga 15 poin (mungkin ada variasi sesuai aturan turnamen).
  • Tim harus menang dua set dari tiga untuk memenangkan pertandingan.
  • Setiap tim diizinkan tiga kali kesalahan servis per set.
3. Rotasi Pemain
  • Pemain melakukan rotasi searah jarum jam setiap kali tim mereka mendapatkan poin untuk menjadi pemberi bola berikutnya.
  • Pemain harus tetap di posisi duduk mereka selama rotasi.
4. Servis
  • Servis harus dilakukan dari tempat duduk pemain.
  • Bola servis harus melewati di atas net dan di antara tiang gawang tanpa menyentuhnya.
  • Pemain yang melakukan servis harus duduk di atas pantat mereka.
5. Bola Hidup
  • Bola dianggap hidup selama masih berada di atas permukaan permainan.
  • Pemain dapat menggunakan bagian tubuh mereka untuk menyentuh atau memainkan bola, termasuk tangan, lengan, dan kepala.
6. Blok
  • Pemain boleh melakukan blok, tetapi mereka harus tetap di atas pantat mereka saat melakukannya.
7. Pertukaran Pemain
  • Pemain dapat digantikan oleh pemain cadangan selama waktu istirahat atau setelah poin.
8. Penalti dan Kesalahan
  • Kesalahan, seperti pelanggaran servis atau sentuhan bola yang tidak sah, akan memberikan poin kepada tim lawan.
  • Pemain yang melakukan kesalahan servis tiga kali dalam satu set akan memberikan poin langsung kepada tim lawan.

Peraturan dapat sedikit bervariasi tergantung pada level permainan dan aturan turnamen tertentu. Oleh karena itu, selalu disarankan untuk merujuk pada aturan resmi yang diberikan oleh organisasi voli terkait atau penyelenggara turnamen.
